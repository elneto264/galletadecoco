Tecnologia usada:
NodeJs

## Instalar

```sh
cd pruebaCeroOne
npm install
npm start
```

Usar ``` npm install``` para instalar los modulos correspondientes, luego usar el 
```sh 
npm start
```
para lanzar la aplicacion desde la terminal
